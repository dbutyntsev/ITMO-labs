# Лабораторная работа №1

**Задание:** 

1. На основе предложенной предметной области составить ее описание. Из полученного описания выделить сущности, их атрибуты и связи.

2. Составить инфологическую модель.

3. Составить даталогическую модель. При описании типов данных для атрибутов должны использоваться типы из СУБД PostgreSQL.

4. Реализовать даталогическую модель в PostgreSQL. При описании и реализации даталогической модели должны учитываться ограничения целостности, которые характерны для полученной предметной области.

5. Заполнить созданные таблицы тестовыми данными.

### Предметная область

Порядок  вещей, каким он существовал, вполне устраивал Шута. Время от времени он мог слегка расстраивать этот порядок, но только едва-едва ощутимо. Он был критиком, а не революционером. На поверхности ровно текущей реки Времени он стремился вызвать лишь легкую рябь. От мысли, что можно изменить и само течение, у него мурашки бежали по коже. Стремление испытать какое-то приключение, кроме тех, что были возможны в сагах, было вытравлено из его сознания так же тщательно и продумано, как и у всех жителей Дисаспара.

### Описание предметной области
Диаспар — это замкнутый город, где жизнь строго регламентирована и неизменна. Главная ценность — сохранение текущего порядка вещей и предотвращение любых радикальных перемен (революций). В обществе существуют специальные роли (например, Шут), которым позволено вносить минимальные искажения («рябь») в поток времени для поддержания психологического баланса жителей.

### Список сущностей и их классификацию: 
**Стержневые сущности:**
1.	Житель: Имя, Роль, Уровень стремления к приключениям.
2.	Город: Название.
3.	Сага: Название, Жанр, Опасность идей (насколько сильно меняет сознание).
Характеристические сущности
4.	Роль: Название, Уровень влияния.
5.	Действие: Название, Уровень влияния.
Ассоциативные сущности:
6.	Течение: Характер, Вид
7.	Изучение саги: Изучается или нет
