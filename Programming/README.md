# Programming Java
Этот раздел репозитория представляет собой хронологию моего профессионального роста от написания базовых консольных скриптов до проектирования **многопоточного распределенного клиент-серверного приложения (Fullstack Java)** со своей базой данных, системой авторизации и графическим интерфейсом. 

### Данный курс я бы разделил на 4 основные части:
Модуль 1: Элементы базового синтаксиса и основы ООП (Лабораторные работы №1–2)

* Работа с примитивными типами данных, ветвлениями, циклами и многомерными массивами. Освоение математических функций стандартной библиотеки java.lang.Math и сборка проекта в исполняемый архив JAR.

* Практическое применение базовых концепций ООП (наследование, полиморфизм, инкапсуляция). Изучение механизмов переопределения методов, областей видимости и подключения внешних библиотек (.jar).

Модуль 2: Проектирование архитектуры и объектное моделирование (Лабораторные работы №3–4):

* Декомпозиция сложных текстовых сценариев в расширяемую иерархию классов в соответствии с принципами SOLID.

* Интеграция в архитектуру современных конструкций языка: перечислений (enum), записей (record) и механизмов обработки исключительных ситуаций (Exceptions).

Модуль 3: Управление коллекциями и сетевое взаимодействие (Лабораторные работы №5–6)

* Проектирование интерактивного CLI-приложения для управления коллекцией TreeSet. Изучение интерфейса Comparable, кастомной валидации многоуровневых моделей данных, работы с файловыми потоками Scanner/FileWriter (формат XML) и обработки рекурсивных скриптов.

* Разделение программы на независимые Клиентский и Серверный модули. Организация асинхронного обмена данными по протоколу TCP с использованием неблокирующих сетевых каналов Java NIO, сериализации объектов и фильтрации данных через Stream API.
